What it replaces

Real tradeoffs. Real numbers.

Not "AI-powered" marketing. Actual replacement comparisons with the tools people already pay for.

vs Deepgram / OpenAI Whisper API $0 / minute

BonfyreTranscribe runs HCP-enhanced Whisper locally. Speaker segmentation, hallucination filtering via Hyperscan (>1 GB/s), filler detection, and eGeMAPSv02 quality scoring (88 audio features). No API key. No per-minute charge. No privacy tradeoff.

vs Strapi / Contentful (CMS) 1,742× smaller

BonfyreCMS is 287 KB — compared to 500 MB for a standard Strapi install with its npm tree. Dynamic schemas, token auth, REST API, full-text search, and structured logging. Zero npm packages. Cold start is instant.

vs Pinecone / Weaviate Cloud 5 ms local search

BonfyreVec + BonfyreEmbed give you NEON SIMD cosine similarity and BM25 hybrid ranking. Embeddings stored in zero-copy LMDB. BonfyreWeaviateIndex connects to self-hosted Weaviate. $70–$250/month hosted → $0/month local.

vs Twilio / LiveKit (realtime) $0 / stream

BonfyreMoQ is a pure C WebTransport/MoQ relay for private realtime audio and media fan-out. No SaaS relay bill, no Node.js in the media path, no per-stream markup. Inline AI scoring via BonfyreTone and BonfyreSpeechLoop.

vs full SaaS backend ~$2,500/mo → $0

BonfyreAuth (BLAKE2b + Ed25519) + BonfyrePay + BonfyreMeter + BonfyreAPI + BonfyreCMS + BonfyreProxy (OpenAI-compatible) + BonfyreGate + BonfyreOffer + BonfyreFinance. All of it. ~2.1 MB total disk. Zero recurring vendor bills.

vs cloud GPU for AI inference Runs on your laptop

FPQ 4× compression means a model that required a $10K cloud GPU runs on a $500 laptop instead. 99.97% cosine similarity preserved. 15 models already published on HuggingFace. KV cache compression gives 4× more context in the same VRAM.

Cloud AI glue Local proxy + planner

Use an OpenAI-compatible local endpoint and a machine-only planner layer instead of wiring every workflow through paid cloud APIs and brittle app glue.

Pipeline flows

How the binaries chain together.

Each binary handles one job. Content-addressed artifacts flow between stages. Re-runs skip unchanged work automatically.

Audio → packaged deliverable

Ingest · Transcribe · Brief · Proof · Offer · Pack · Distribute

Audio normalized by BonfyreIngest or BonfyreMediaPrep → HCP transcription with speaker segmentation → structured brief → BLAKE2b proof bundle → dynamic pricing → packaged artifact → QUIC distribution or swarm delivery. 5–8 ms per stage.

Recording → searchable knowledge

Transcribe · TranscriptClean · Paragraph · Embed · Vec · Index · Query

Raw recording → clean transcript → paragraphs → sentence embeddings → SIMD vector index → hybrid BM25+cosine search. BonfyreTranscriptFamily clusters related recordings into a searchable family. Everything stored in zero-copy LMDB.

Content → published pages

Brief · Repurpose · Render · Emit · Sync

Brief generated by BonfyreBrief → BonfyreRepurpose extracts multi-format content (show notes, social clips, email copy) → BonfyreRender builds HTML → BonfyreEmit produces PDF/EPUB/JSON → BonfyreSync publishes to your target.

Work → revenue

Proof · Offer · Finance · Pay · Meter · Ledger · Outreach

Content-addressed proof bundle → dynamic offer pricing via BonfyreOffer → BonfyreFinance calculates margins → BonfyrePay handles checkout → BonfyreMeter tracks usage → BonfyreLedger maintains the audit trail → BonfyreOutreach sequences follow-up.

Repo → onboarding guide

Ingest · Tag · Canon · Brief · Graph · Render · Emit

Source tree ingested → BonfyreTag extracts topics → BonfyreCanon deduplicates → BonfyreBrief generates structured overview → BonfyreGraph maps relationships → BonfyreRender produces an HTML onboarding guide → BonfyreEmit publishes it.

Pipeline execution

BonfyrePipeline · Queue · Orchestrate · Runtime

BonfyrePipeline runs stages with hot-load swap (32-stage registry, 16 A/B pairs). BonfyreQueue buffers work. BonfyreOrchestrate manages multi-pipeline flows. BonfyreRuntime is the shared execution environment for in-process coroutine paths.

Who it fits

Bonfyre fits anywhere raw input needs to become trusted, reusable output.

Legal and professional services

Private intake → structured briefs

Client calls transcribed locally (zero cloud exposure), structured by speaker, quality-scored, and packaged into proof bundles with BLAKE2b provenance. Ed25519 signatures for audit trails. HIPAA-compliant by default — nothing leaves the building.

Media and podcasting

Raw audio → published, monetized asset

Episode in → transcript + show notes + social clips + RSS feed + checkout page, all locally in under 5 minutes. BonfyreClips, BonfyreNarrate, BonfyreStitch, BonfyreRepurpose, BonfyreOffer in one pipeline. No Descript. No GPT-4 API key.

Field research and archives

Offline capture → searchable archive

Record in the field with no internet. Bonfyre transcribes on-device, embeds for SIMD vector search, creates a LMDB-backed searchable archive, and publishes a static site. Fountain codes handle lossy syncs via BonfyreDistribute.

Ops and internal teams

Meetings → actionable memory

Standups, postmortems, shift handoffs, and town halls — transcribed, summarised, tagged, and made searchable. BonfyreProject tracks follow-up items. BonfyreGraph links related records. Teams can find last month's incident in seconds.
