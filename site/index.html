What Expanded

The stack now covers more than compression and pages.

Recent work pushed Bonfyre deeper into serving, transport, zero-copy reads, in-process pipeline execution, and the product surfaces that sit on top of them.

API runtime

Per-core serving

Shared-nothing reactor loops, faster HTTP parsing, SSE support, and per-reactor heaps where mimalloc is available.

Pipeline runtime

Hot-load and coroutine flow

Swap stages, route A/B traffic, and run coroutine paths in process instead of rebuilding the whole pipeline around every change.

Transport

QUIC family streams

Artifact families move over multiplexed streams with priority delivery and reconnect behavior built for sync-heavy flows.

P2P distribution

Swarm delivery

BitTorrent v2 path for artifact bundles, tied back to Bonfyre hash, gate, and meter instead of bolted on beside them.

Zero-copy data path

Read-hot cache

LMDB and FlatBuffer manifests cut repeated parse and lookup overhead where the same artifacts get touched again and again.

Newer surfaces

Proxy, orchestration, tel, and verified audio

bonfyre-proxy, bonfyre-orchestrate, bonfyre-tel, bonfyre-moq, bonfyre-swarm, and bonfyre-narrate extend the stack into local AI access, planning, communications, relay, delivery, and finished output.
