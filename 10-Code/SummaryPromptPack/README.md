# Summary Prompt Pack

Reusable prompt templates for generating different summary styles from the same transcript.

## Run

```bash
python3 prompts.py --list
python3 prompts.py --template executive-summary --transcript transcript.txt
```

## Templates
- `executive-summary` — decisions + next steps for founders
- `meeting-notes` — full meeting recap with attendees
- `student-notes` — learning-focused summary with key concepts
- `creator-repurpose` — quotable moments + content angles
