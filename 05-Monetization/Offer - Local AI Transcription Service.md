# Offer: Local AI Transcription Service

## Offer
- who it is for: founders, creators, students, and anyone with useful audio they never process
- painful problem: recordings pile up and never become usable notes
- promised outcome: clean transcript, concise summary, and action items from one audio file
- turnaround: same day or next day for small files

## Deliverable
- raw transcript
- 5-bullet summary
- action items

## Pricing
> Full pricing research and evidence: [[03-Research/Research - Transcription Pricing Analysis]]

| Tier | File Length | Price |
|---|---|---|
| Quick | Under 15 min | $12 |
| Standard | 15-45 min | $18 |
| Deep | 45-90 min | $25 |
| Complex | Multi-speaker / 90+ min | $35 |

- start price for cold outbound: $15 (standard tier, discounted to test conversion)
- compute cost per file: $0.00 (local M3 hardware)
- estimated labor per file: 8-12 min active time
- effective hourly rate at $18 standard: $108/hr
- batch discount: 15% for 3-pack, 25% for weekly 5-pack

## Sales Angle
- headline: "Send me one messy recording. Get back a clean transcript, 5-bullet summary, and your action items. $15 for anything under 45 min. Done today."
- anchor: human transcription costs $45+ on Rev for just the transcript — we include summary + action items for 60% less
- trust signal: runs 100% on local hardware, your audio never touches a server
- call to action: send one file and get back a structured deliverable same-day
- first-file hook: offer one free demo file to best prospects to prove quality

## Delivery Workflow
1. receive audio file
2. transcribe locally
3. review and format summary
4. return transcript, summary, and actions

## Risks
- transcript quality may vary on poor audio
- turnaround promise must match real processing speed

## Pricing Validation Plan
- [ ] Price first 3 outbound messages at $15 (standard tier)
- [ ] Offer one free demo file to the best prospect
- [ ] Track: conversion rate, pushback frequency, willingness-to-pay signals
- [ ] If conversion >30%, test $20 on next batch
- [ ] If conversion <10%, drop to $12 with first-file-free hook
