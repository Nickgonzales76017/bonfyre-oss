# Concept: Network Effects

## Summary
Network effects appear when each new participant, user, or node makes the system more valuable for the next one.

## Why It Matters
- some ideas in the vault get stronger with density, not just effort
- it helps distinguish linear businesses from compounding networks

## Where It Applies
- crowdsourced tasks
- local logistics nodes
- decentralized infrastructure ideas
