# Idea: Local Bootstrap Kit

## Summary
Create a one-command or low-friction setup kit that gets a new machine ready for the local transcription workflow.

## Core Insight
Build friction compounds when environment setup is tribal knowledge instead of a repeatable kit.

## First Use Case
Install dependencies, verify tools, create output folders, and print the next steps for a new environment.

## Why It Might Work
- inefficiency: local setup friction slows every restart
- arbitrage: a bootstrap kit turns setup into infrastructure
- trend: tiny internal developer tools create outsized leverage
