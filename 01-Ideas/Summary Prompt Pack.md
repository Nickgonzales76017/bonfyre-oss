# Idea: Summary Prompt Pack

## Summary
Create reusable summary/action-item prompt templates tuned for different buyer types and recording contexts.

## Core Insight
The same transcript needs different summaries for founders, creators, students, and operations use cases.

## First Use Case
Generate alternate outputs for a single transcript: executive summary, meeting notes, student notes, and creator repurposing.

## Why It Might Work
- inefficiency: one-size-fits-all summaries feel generic
- arbitrage: prompt specialization increases perceived value
- trend: tailored output formats matter more than generic AI output
