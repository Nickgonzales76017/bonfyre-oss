# Idea: Quality Scoring Loop

## Summary
Create a lightweight evaluation loop that scores transcript and summary quality so improvements are based on evidence, not guesswork.

## Core Insight
Without a scoring loop, quality upgrades blur into vibes and are hard to compare.

## First Use Case
Rate transcript usefulness, summary usefulness, and action-item relevance for the first few jobs.

## Why It Might Work
- inefficiency: quality changes are hard to compare informally
- arbitrage: feedback loops improve the service faster
- trend: evaluation is becoming core to useful AI systems
